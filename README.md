# Previsão de Renda

*Desenvolvido na Formação: Cientista de Dados – EBAC*

## Descrição

Este foi o segundo projeto da formação, utilizando um conjunto de dados de previsão de renda disponibilizado pelo Kaggle, com o objetivo de realizar análise exploratória completa e construir um modelo de classificação para prever o risco de inadimplência (“mau pagador”). O projeto envolveu uma análise detalhada dos dados, incluindo estatísticas descritivas, análise univariada e bivariada, correlação entre variáveis e criação de variáveis derivadas (como faixas de idade e tempo de emprego). 

Foram utilizadas ferramentas como Python, pandas, seaborn, Matplotlib, scikit-learn e ydata-profiling para limpeza, transformação, visualização e preparação dos dados. Além das ferramentas de machine learning, o projeto também incorporou Streamlit para criação de uma interface interativa e VSCode para desenvolvimento do protótipo, permitindo uma visualização clara dos resultados e facilitando a apresentação da análise.

Para a modelagem supervisonada, foi treinado e avaliado um modelo utilizando RandomForestClassifier, com divisão treino/teste, tratamento de variáveis categóricas por meio de one-hot encoding e análise de métricas como acurácia, matriz de confusão e relatório de classificação.

O trabalho proporcionou experiência prática na preparação de dados reais, implementação de modelos preditivos, criação de dashboards e no uso integrado de ferramentas do ecossistema de ciência de dados em um fluxo completo de análise.
